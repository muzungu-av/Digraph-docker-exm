# Кластер Dgraph

Production-ready отказоустойчивый кластер Dgraph v25.1.0, с учётом:

3 узла Zero (для отказоустойчивости Raft),
2 узла Alpha (с репликацией данных),
1 Ratel,
ACL с токеном,
фиксированные IP-адреса (для стабильности),
корректные порты и флаги для v25.1.0.

Пароль по умолчанию для Groot — password.

## Графовая база данных Dgraph

Графовая база данных Dgraph для хранения связей между узлами, пользователями, транзакциями и ролями.
И визуализиция графа через веб-интерфейс Ratel.
